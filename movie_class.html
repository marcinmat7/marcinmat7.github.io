Movie classification¶

In this short notebook I'll present a very simple way of how to classify the genre of the movie based on its description. We will build a simple model based on tf-idf vectorizer. Firstly, we will scrap the data from imdb.com website. Then, we'll make some pre-processing steps. In the end we will build the models and compare the performance for train and test datasets. Lets get it started!

Data scrapping¶

In order to scrap the data, lets take a look at the hypoerlinks for the movies. From the list Top Box Office we can select a specific genre and select the 50 titles using the folowing link:

https://www.imdb.com/search/title/?genres=action&start=51&explore=title_type,genres&ref_=adv_prv

It shows 50 action movies with their description. The hyperlink showing next 50 movies is pretty much the same (only the number in the middle of the string has to be changed):

https://www.imdb.com/search/title/?genres=action&start=101&explore=title_type,genres&ref_=adv_nxt

So, we can easily automate generating the hyperlinks via regular expressions. For the Action movies we will use the following link,

https://www.imdb.com/search/title/?genres=action&start=|no|&explore=title_type,genres&ref_=adv_nxt

detect the pattern |no| and replace it with "51", "101", "151" and so on. For the other genres I prepared the following table:

It turns out that detecting the part of the code with the movies description is relatively easy as well. All we have to do is to detect two strings <p class=\"text-muted\"> and '< / p > \ "' (without spaces). Text between these patterns is the description of our movie. The label is of course depended on the link.

In [3]:
desc = []
genre = []
    
for j in range(0, len(links)):
    for n in range(1,100):        
        start = 50 * n + 1   
        url = links['Link'][j]
        url = url.replace("|no|", str(start))
        page = urlopen(url)
        html_bytes = page.read()
        html = html_bytes.decode("utf-8")
        html2 = html.split("loadlate=")
        html2 = html2[1:51]
    
        for i in range(len(html2)):
            tmp = html2[i]

            x = re.search("<p class=\"text-muted\">", tmp)
            tmp2 = tmp[(x.span()[1]+1):]
            x = re.search("</p>", tmp2)
            tmp3 = tmp2[:x.span()[0]]
            desc.append(tmp3)
            genre.append(links["Genre"][j])

Building the model¶

In [14]:
def split_fit_eval(X, y, model=None,
                   epochs=15,
                   random_state=0):
    X_train, X_test, y_train, y_test = \
    train_test_split(X, y, random_state=random_state)
    
    if not model:
        model = Sequential()
        model.add(Dense(3, input_dim=X.shape[1],
                        activation='sigmoid'))

        model.compile(loss=tf.keras.losses.CategoricalCrossentropy(),
                      optimizer='adam',
                      metrics=['accuracy'])
    
    h = model.fit(X_train, y_train,
                  epochs=epochs,
                  verbose=1, 
                  validation_split=0.2)
    
    loss, acc = model.evaluate(X_test, y_test)
    
    return loss, acc, model, h
